# [05] ARCHIVE_SCREEN: SYSTEM_DATABASE (FINAL SPEC)

## I. NAVIGATION ARCHITECTURE
- **LEFT PANE**: Hierarchical directory tree (`[DIR]`).

  - `MANIFEST/`: The Collapse, Sector Records, Credits.
- **RIGHT PANE**: Content viewer with static metadata header (FILE, SIZE, PERM).

### 2. HANDBOOK [FINALIZED]

### 3. MANIFEST [FINALIZED]
- **THE_COLLAPSE**: Chronological scrolling lore.
- **SECTOR_RECORDS**: Modular readout of Max Wave, Lifetime Purge, and Rank.
- **CREDITS**: System Attributions for Architect, Engine, Visuals, and Audio.
