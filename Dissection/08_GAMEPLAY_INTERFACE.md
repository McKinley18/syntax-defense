# [08] GAMEPLAY_INTERFACE: HUD_&_CONTROLS

## I. VITALS_OVERLAY (TOP_LEFT)
- **KERNEL_INTEGRITY**: Current HP (0-20).
- **TOKEN_RESERVE**: Active Credits/Currency.
- **WAVE_STATUS**: Wave Number and Active Status.

## II. TOWER_DASHBOARD (BOTTOM)
- **TOWER_CARDS**: Click to select/purchase.
- **COST_LABELS**: Red font if insufficient credits.
- **LEVEL_GATES**: Unlock visibility based on rank.

## III. CONTEXT_MENU (ON_SELECTION)
- **POSITION**: Appears above the dashboard on selection.
- **UPGRADES**: Damage, Speed, Range scaling.
- **REMOVAL**: Partial credit refund (50%).
- **AUTO_UPGRADE**: Sequential tiering.
