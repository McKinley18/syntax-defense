# [03] MAIN_MENU: SYSTEM_ROOT

- **PRIMARY_NAV**:
  - `INFILTRATE_CORE` -> `MODES` screen.
  - `SYSTEM_ARCHIVE` -> `ARCHIVE` screen.
  - `DIAGNOSTICS` -> `SETTINGS` screen.

## II. DYNAMIC_OVERLAY
- **UPTIME**: Incremental session timer.
- **ENTROPY**: 0.14 base value with randomized jitter (0.00 - 1.00).

- **BACKGROUND**: Animated grid with sweep/comet effects.
- **INTERACTION**: `mousePos` tracked for menu parallax.
